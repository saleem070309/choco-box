# Choco Box Web Application 🍫

A premium e-commerce landing page for Choco Box, built with FastAPI (Python) and a modern glassmorphism frontend.

## Features
- 🌟 Premium & Luxurious Design.
- 📱 Fully Responsive (Mobile-Friendly).
- 🛒 WhatsApp Order Integration.
- 🔍 Fuzzy Search for Products.
- 🖼️ Dynamic Image Gallery.
- 🛠️ Admin Panel for Order Management.

## Tech Stack
- **Backend:** FastAPI (Python)
- **Database:** SQLite
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons:** Phosphor Icons
- **Animations:** ScrollReveal.js & Swiper.js

## Local Setup
1. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Linux/Mac
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   bash run.sh
   ```

## Deployment
This project is ready to be deployed on platforms like **Render**, **Railway**, or **Heroku**.
- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python api/main.py`
